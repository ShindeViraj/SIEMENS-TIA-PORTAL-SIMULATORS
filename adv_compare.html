OUT_RANGE (Value outside range)

Instruction Description: Tests whether an input value is outside of a specified value range. If VAL < MIN or VAL > MAX, the output is TRUE.

How it Works:
  • Logic: The output turns TRUE only when the input VAL is strictly less than MIN or strictly greater than MAX.
  • Application: Useful for triggering alarms when a process variable (like temperature or pressure) deviates outside a safe operating window.

OK / NOT_OK (Validity Check)

Instruction Description: Tests whether an input data reference is a valid real number (IEEE 754). It is invalid if it is NaN (Not a Number), +/- Infinity, or denormalized.

How it Works:
  • NaN (Not a Number): Results from undefined operations like 0.0 / 0.0 or the square root of a negative number.
  • Infinity: Results from operations like dividing a non-zero number by zero.
  • Denormalized: Very small numbers close to zero that exceed standard floating-point precision bounds. The CPU treats these as zero in calculation.
